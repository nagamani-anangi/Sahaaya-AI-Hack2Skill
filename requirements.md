# requirements.md

## Functional Requirements
- User profile input (age, education, income, category, location)
- Government scheme data storage
- AI-based eligibility and recommendation logic
- Multilingual output (Telugu, Hindi, English)
- Chat-style user interaction
- Scheme application guidance

## Non-Functional Requirements
- Easy to use interface
- Low internet data usage
- Secure handling of user data
- Scalable system design

## Constraints
- Uses publicly available government scheme data
- Requires basic internet connectivity
